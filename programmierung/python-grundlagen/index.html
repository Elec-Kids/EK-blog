Was du mit Programmierung und Mikrocontrollern lernst

Mit Programmierung und Mikrocontrollern lernst du mehr als nur Code. Du trainierst logisches Denken, Problemlösen, Geduld und den Umgang mit Technik.

Thonny für Eltern: Einstieg in Python und MicroPython

Thonny ist eine einfache Entwicklungsumgebung für Python und MicroPython. Diese Anleitung zeigt Eltern, wie sie Thonny installieren, erste Programme starten und einen Mikrocontroller verbinden.

Was ist ein Mikrocontroller?

Ein Mikrocontroller ist ein kleiner Computer für bestimmte Aufgaben. Er liest Eingaben, verarbeitet sie und steuert damit LEDs, Sensoren, Displays oder andere Bauteile.
